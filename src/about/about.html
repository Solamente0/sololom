About Sololom

Sololom is a browser extension that provides seamless access to Large Language Models (LLMs) directly in your browser. With Sololom, you can chat with various AI models without leaving your current page.

Features

  • Multiple Chat Modes: Choose between popup and full-page interfaces
  • Model Selection: Support for OpenAI, Anthropic, and Mistral models
  • Customizable Settings: Configure global preferences and chat-specific options
  • Conversation Management: Save, export, and manage your chat history
  • Keyboard Shortcuts: Quick access to Sololom's features

Getting Started

  1. Add your API keys in the Settings
  2. Choose your preferred model
  3. Start chatting using the popup or full-page mode

Privacy

Sololom is designed with privacy in mind. Your conversations and API keys are stored locally in your browser. No data is sent to our servers. All communications happen directly between your browser and the LLM providers.

Frequently Asked Questions

How do I get an API key?

You need to create an account with the LLM provider of your choice and generate an API key from their developer portal:

  • OpenAI API Keys
  • Anthropic API Keys
  • Mistral API Keys

Is my data private?

Sololom stores your conversations and settings locally in your browser. We don't collect or transmit your data. However, when you interact with LLM APIs, your messages are sent to those providers according to their privacy policies.

How can I reset all settings?

Go to the Settings page and click the "Reset All Settings" button at the bottom of the page.
